Principais formatações

Negroto / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semantica)

Nesta frase, tenho um termo em destaque usando a tag <strong>(essa é a forma correta!)

Itálico / Enfase

Nesta frase, temos um termo em itálico usando a tag <i>(não semantica)

Nesta frase temos um termo em enfase usando a tag <em>(semântica)

Texto marcado

Podemos criar também, um texto marcado usando a tag <mark>

Texto Grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. usa a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. usa a tag <ins>

Texto sobrescrito

para inserir coisas do tipo x20+3 usa a tag <sup>

Texto subscrito

Para coisas do tipo H2O usa a tag <sub>
